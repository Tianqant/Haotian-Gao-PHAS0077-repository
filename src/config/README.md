Here is a non-exhaustive list of all the options you can add to the config files in order to construct an experiment.

#### Experiment

- ***Version***
- ***Description***
- ***Path***
- ***Seed***

#### Data

- ***Type***
- ***args***

    - ***Reduced***
    - ***Validation Size***
    - ***Customer Transforms***
    - ***Product Transforms***

#### Model

- ***Type***
- ***Loss***
- ***args***
